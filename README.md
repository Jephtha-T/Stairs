Where

Interactive Three.js project built around a `Stairs.glb` model, optical alignment, shifting shadows, and mouse-controlled viewing.

Interactive Three.js project creating a digitalised art piece mainly inspired by a piece made by ByungJoo Kim Titled "Ambiguous Wall-Stairs YB1" From his Ambiguous Parallax Exhibition in Seoul 2022.  

## Run Locally

Install dependencies once:

```bash
npm install
```

Start the Vite dev server:

```bash
npm run dev
```

Open the local URL that Vite prints, usually:

```text
http://127.0.0.1:5173/
```

## Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
